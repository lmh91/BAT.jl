List of algorithms

BAT.jl offers performant implementations of mutliple algorithms for sampling, integration and optimization.

Sampling algorithms

All following sampling algorithms can be passed to bat_sample:

samples = bat_sample(sampleable, sampling_algorithm).result

IIDSampling

BAT.jl sampling algorithm type: IIDSampling

sampling_algorithm = IIDSampling(nsamples=10^5)

Metropolis-Hastings

BAT.jl sampling algorithm type: MCMCSampling MCMC algorithm subtype: MetropolisHastings

sampling_algorithm = MCMCSampling(mcalg = MetropolisHastings(), nsteps = 10^6, nchains = 4)

Hamiltonian MC

BAT.jl sampling algorithm type: MCMCSampling MCMC algorithm subtype: HamiltonianMC

sampling_algorithm = MCMCSampling(mcalg = HamiltonianMC())

Reactive Nested Sampling (experimental)

BAT.jl sampling algorithm type: ReactiveNestedSampling

sampling_algorithm = ReactiveNestedSampling()

Ellipsoidal Nested Sampling (experimental)

BAT.jl sampling algorithm type: EllipsoidalNestedSampling

sampling_algorithm = EllipsoidalNestedSampling()

Partitioned Sampling (experimental)

BAT.jl sampling algorithm type: PartitionedSampling

sampling_algorithm = ReactiveNestedSampling()

Sobol Sampler

BAT.jl sampling algorithm type: SobolSampler

sampling_algorithm = SobolSampler(nsamples=10^5)

Grid Sampler

BAT.jl sampling algorithm type: GridSampler

sampling_algorithm = GridSampler(ppa=100)

Prior Importance Sampler

BAT.jl sampling algorithm type: PriorImportanceSampler

sampling_algorithm = GridSampler(nsamples=10^5)

Integration algorithms:

All following integration algorithms can be passed to bat_integrate:

integral = bat_integrate(sampleable, integration_algorithm).result

Adaptive Harmonic Mean Integration (AHMI)]

BAT.jl integration algorithm type: AHMIntegration 

integration_algorithm = AHMIntegration()

Vegas Integration

BAT.jl integration algorithm type: VEGASIntegration 

integration_algorithm = VEGASIntegration()

Cuhre Integration

BAT.jl integration algorithm type: CuhreIntegration 

integration_algorithm = CuhreIntegration()

Divonne Integration

BAT.jl integration algorithm type: DivonneIntegration 

integration_algorithm = DivonneIntegration()

Integration via Bridge Sampling (experimental)

BAT.jl integration algorithm type: BridgeSampling 

integration_algorithm = BridgeSampling()

Mode finding algorithms:

All following mode finding algorithms can be passed to bat_findmode:

mode = bat_findmode(sampleable, modefinding_algorithm).result

Nelder-Mead Optimization

BAT.jl mode finding algorithm type: MaxDensityNelderMead
